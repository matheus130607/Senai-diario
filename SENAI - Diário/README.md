# somativa-diario

Projeto consolidado a partir de variantes do "SENAI - Diário". Aplicação front-end em React + Vite.

Como rodar

```bash
npm install
npm run dev
```

Build

```bash
npm run build
```

Estrutura importante

- `index.html` — entrada HTML
- `src/` — código fonte React
- `public/` — ativos públicos (favicon, robots.txt)

Observações

- Este repositório foi gerado consolidando várias versões; alguns arquivos de configuração foram adicionados manualmente.
